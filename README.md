# Venue Management System

A simple web app to **manage event venues**.  
Add, view, update, and delete venues with details like name, location, capacity, date, and availability.

## Tech Stack
- **Backend:** Spring Boot, JPA, MySQL  
- **Frontend:** React, Axios, Tailwind CSS  
- **Database:** MySQL
